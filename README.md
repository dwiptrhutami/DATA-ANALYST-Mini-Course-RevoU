# Students Performance in Exams — Data Analysis Project

📌 Overview
This project analyzes the Students Performance in Exams dataset from Kaggle to explore factors that influence students’ scores in three subjects: Mathematics, Reading, and Writing.
The dataset includes demographic and background information such as gender, parental education level, lunch type, and test preparation course.

📊 Dataset source:
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

🎯 Objectives
The main goals of this project are:

1. To explore score distributions across subjects
2. To analyze the relationship between gender and exam scores
3. To identify key factors affecting academic performance
4. To perform Exploratory Data Analysis (EDA)

📂 Dataset Information
The dataset contains 1000 student records with the following features:

gender
race/ethnicity
parental level of education
lunch
test preparation course
math score
reading score
writing score

🧪 Methods Used

The following techniques were applied:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Data Visualization (Matplotlib, Seaborn)
- Categorical Encoding (e.g., gender_encoded)

📊 Key Insights
Gender shows a weak correlation with exam scores:
Math score: weak negative correlation
Reading score: weak positive correlation
Writing score: weak positive correlation
No strong relationship is found between gender and academic performance
Writing scores show slightly clearer variation across groups compared to other subjects

📈 Example Visualizations
https://datastudio.google.com/reporting/8ff097c3-76e0-4807-ba0f-d8c364b6509e

📌 Conclusion
The analysis suggests that gender has only a weak influence on students’ exam performance. Other factors such as parental education level and test preparation tend to play a more important role.
